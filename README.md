# DeepSeek Auto Next

Automatically continues DeepSeek responses when generation pauses. Perfect for long outputs and scripts.

## Features

- Auto triggers "Continue" when generation stops
- Seamless long-form output
- Lightweight and minimal
- No tracking, no external servers

## Installation (Chrome)

1. Download the latest ZIP from Releases
2. Extract the ZIP
3. Open chrome://extensions
4. Enable Developer Mode (top right)
5. Click "Load unpacked"
6. Select the folder containing manifest.json

## Usage

Open DeepSeek.
Start generating a long response.
The extension will automatically continue when output pauses.
